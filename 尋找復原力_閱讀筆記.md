# 📚 讀書筆記：《尋找復原力》

**閱讀日期**：2026-04-02  
**主題**：心理復原力、情緒管理、自我成長、人際關係  
**核心概念**：在逆境中彈回的能力，做自己最強大的守護者。

---

## 🌱 Part 1：在逆境中，建立復原力

### 核心思想
- 生命下一刻無法掌控，在挫敗後建立復原力，讓心理傷口癒合。
- 不斷自我探索與成長，有助自己建立復原力。

### 復原力的培養
- ✅ 給自己情緒一個空間，接納自己。
- ✅ 每個新開始須要勇氣。
- ✅ 花時間進入自己的內心。
- ✅ 找個人去倚賴、支持 → **人與人的連結**。

### 六項復原力技能
1. **覺察**：看見及面對挫敗。
2. **調節**：管理情緒與身體反應。
3. **心智敏捷**：靈活思考，不被僵化思維困住。
4. **樂觀**：以積極角度解讀困境。
5. **效能**：相信自己能夠辦到。
6. **連結**：建立健康的人際關係。

> **重點**：建立復原力，就是在鍛鍊心智。

## 🌈 Part 2：復原力在情緒和身體裡

### 情緒觀
- 任何情緒出現時，都是被歡迎，待在自身內，就有在家的感覺。
- 生命就是 **快樂 + 痛苦**。
- 每種情緒沒有對錯，逃離痛苦情緒，就失去與痛苦情緒共處的機會。
- 讓自己變成可容納每種氣候的**天空**。
- **疼愛自己，溫柔地和自己一起**。

### 面對哀悼
- 見證及陪伴他人、學習失去。
- 擁有後會失去，所以獲得與失去是兩面，失去後更知擁有是如此可貴。
- 在哀悼中感受喜悅，兩者同時都須要給予空間。
- **如何看待死亡，就會知如何看待生命**。

### 愧疚與接納
- 好好傾聽內心的愧疚：知道自己做錯、傷害了別人。
- 內心道歉、修正行為、彌補過失。
- 但其實，**沒有真正對錯的事**，理解及認識自己的價值觀 → 接納自己並不完美。
- 有缺點和局限，須要休息及建立界線。
- **採取行動，把愧疚轉化為感激**。

### 歡迎那些不被歡迎的
- 讓抱怨好好存在。
- 向人抱怨/聽人抱怨是生活中常遇見的事情，抱怨需要的，是一個空間。
- 與抱怨好好相處，看見自己的脆弱。
- **歡迎那些不被歡迎的**：痛苦、脆弱、羞愧、不完美。

### 傾聽身體的聲音
- 世界越快速、越混亂時，我們更須慢下來：**呼吸、說話、回應**。
- 每天給自己暫停時間去調節自己神經系統。
- **向孤獨靠近，練習和自己相處**。
- 面對孤獨 → 逃避/靠近。
- **獨處**：閱讀、思考、寫作、散步、欣賞大自然 → 自我充電。
- 要走入內心世界，先讓自己慢下來。
- **改變你和討厭部分之間建立溫柔的係**，靠近、理解、去接納。

## 🧠 Part 3：復原力在大腦裡

### 面對失敗
- 面對失敗 → 這是**挑戰**，還是**威脅**？
- 如何看待失敗，來自於成長過程。
- 建立復原力的因素：從不同角度看待失敗、把每次挫敗視為挑戰。
- **失敗是很好、很正常**，因可以從失敗中學習。
- 建立**成長心態**，提升復原力。
- 對成功或失敗本身並不重要，每次失敗是學習，跨出舒適圈，延伸擴張自己能力。
- 固化心態的人，會把別人的成功視為威脅。
- **停止比較**，將別人視為觀點來源，向他人學習。

### 大腦編劇故事軸（五種）
1. **讀心術**：以為別人正如自己這樣想法，不去問/澄清/溝通 → 破壞雙方關係。
2. **都是因為我**：產生自我內疚、差愧、悲傷 → 封閉自己而不與人溝通。
3. **都是別人**：產生憤怒而作出衝動行為，阻礙與人溝通。
4. **災難要來了**：更感焦慮和恐懼，把精力放在糟糕後果，就會力氣解決問題。
5. **做什麼都沒用**：讓人感無望，無助。

### 思考彈性
- 你是想法，還是看見想法的人？
- 復原力來自於**思考彈性**，讓僵化慣性思考鬆動。
- **樂觀**，是可以學習。
- 困境視為挑戰。
- 朝困境走去，著手調整可改變的事。

### 當人生梯子被拿掉
- 把梯子換成**方格攀爬架** → 有阻擋時，只需轉個彎，換個方向就好，每個方向都有路，有新的可能。
- 跳過懸崖，從熟悉到未知，唯有相信，就能跳得過去。
- 每一個第一次，都是在嘗新事物，人生就是有無限種可能。

### 善用自己的優點
- 相信自己能夠辦到。
- 檢示自己優點和長處並善用。

## 🤝 Part 4：復原力在關係裡

### 關係智商
- 建立關係智商（人與人連結的能力），提高復原力。
- 關係中如何與人相處：**要求、拿取、接受、給予、分享、拒絕、玩樂或想像**。

### 關係決裂
- 當關係決裂，光才能照進，檢視關係決裂事件：**要連結/要贏**？
- 學習真誠道歉。
- 為自己的部分道歉。
- 謝謝別人對自己的道歉。
- 當好事發生時，真心意地替對方感到開心。

### 好好溝通
- **如何聽比如何說更重要**。
- 同理傾聽，好好接住對方傾出來的東西。
- 放下自己的看法和評價。
- 別人能和自己撐住痛苦和悲傷，這些重量就會變輕一點，更有能力和自己痛苦待在一起。

## 🌟 Part 5：當危機過後，你要帶走什麼？

### 尋找意義
- 經歷挫敗後，讓自己活回來。
- 從失敗與困境中，尋找意義。
- 「尋找意義」：擁有最終選擇如何回應。
- 在生命中，讓自己「現身」：每次面對挑戰，就算結果失敗，還是讓自己去嘗試；即使會心碎或被拒絕，亦坦然面對自己的內心世界。

### 成長與改變
- 成長，需要讓自己能夠待在「之間」：經歷混亂 → 瓦解舊我 → 留強棄弱 → 建立新我。
- **混亂，是改變的開始**。
- 學習向生命「投降」。
- 放掉控制，擁抱未知。

### 價值觀是指南針
- 價值觀是你的北極星，幫助你指引方向。
- 過一個符合價值觀的人生。
- 每一天，你要對什麼說「好」和「不好」？
- 不能控制生命的時間，但可控制如何活出生命。
- 「你是誰」也不斷改變，成為你想要的人。

## 📝 個人反思與行動清單

### 我學到了什麼？
- [ ] 接納自己的不完美，溫柔地和自己一起。
- [ ] 將失敗視為挑戰，而非威脅。
- [ ] 建立健康的人際連結，尋求支持。
- [ ] 每天給自己暫停時間，調節神經系統。
- [ ] 練習獨處，與自己相處。
- [ ] 用成長心態看待別人的成功。
- [ ] 在關係中真誠道歉，並接受別人的道歉。
- [ ] 尋找生命中的意義，即使面對挫折。

### 我想實踐的行動
1. **每天**：花 10 分鐘獨處，閱讀、思考或散步。
2. **每週**：反思一次「大腦編劇故事」，檢查有無陷入負面思維。
3. **當失敗時**：問自己「這是一次學習機會嗎？」，而非「我為什麼這麼差？」。
4. **當關係決裂時**：先問自己「我是想連結，還是想贏？」。
5. **當感到孤獨時**：主動向信任的朋友傾訴，或寫下自己的感受。
